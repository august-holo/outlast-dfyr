Player Turn
Each player gets 3 actions per turn plus 1 extra action for each additional meeple they have. 

A meeple spawns at the settlements

Actions include:
Move – Travel to an adjacent tile and reveal it if it is face down.
Gather Resources – If a tile has resources, spend 1 action per resource to collect them.

If there is no wolf on the tile the meeple may
Hunt Prey – If a tile has prey, spend 1 action and roll an 8-sided die (d8):
5-8 → Success (You catch prey).
1-4 → Failure (You miss).

If a wolf in on the same tile as a meeple, the meeple must take one of two actions on the the turn. If you have already used all your actions, the meeple on the tile with the wolf will be forced to fight the wolf as though it were an action

Fight a Predator – If a tile has a wolf, roll a d8:
7-8 → Victory (You defeat the predator).
1-6 → Death (You do not survive).

Hide – Use 1 action to avoid predator attacks

Build: Spend 3 resources to construct something.

Drop Resource: 
if you drop a resource/prey on a full tile, the resource/prey will go to a free adjacent tile

Fight another player:
If you are on the same tile as another meeple, you can choose to fight them
both players roll d8, the higher number wins
both players can add resources to change their roll

End of turn:

Meeple consume food, one prey per meeple
If there is no prey for a meeple to consume, they starve are are removed from the board.

Board Turn

The board takes a turn after all players have acted. The following events occur:
Resources Grow – Resources regenerate, but only if at least one resource remains. If a tile has 0 resources, it will not regenerate. A tile can hold a maximum of 3 resources.
Prey Multiply – If 2 or more prey (like rabbits) are on a tile, one more appears (max 3 per tile). If only 1 prey is left, it does not multiply. If there are 3 prey, a new prey is added to an adjacent tile and subsequent prey will be added to the same  tile until 3 prey are present.
Wolves Move – Wolves move toward the nearest tile with prey.
Wolves Multiply – If a tile has 3 prey and 1 wolf, a new wolf appears. However:
There can only be 1 wolf per tile.
The new wolf moves to an adjacent tile with the largest number of prey.
New People Arrive – Each player gets one new meeple in their group.
Wolf Attacks – If a player is on the same tile as a wolf and didn’t hide, the wolf attacks.
Once the board turn is complete, a new round begins with the player turn phase.
